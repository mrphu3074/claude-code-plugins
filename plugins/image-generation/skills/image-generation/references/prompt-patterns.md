# Prompt Patterns for Image Generation

## UI/UX Design Patterns

### Dashboard Design
```
Modern [style] dashboard interface with [theme] theme, displaying [data type]
through [chart types], featuring [navigation elements], [color scheme] color palette,
[additional UI elements]
```

**Examples:**
- "Modern minimalist dashboard with dark theme, displaying sales analytics through line charts and bar graphs, featuring sidebar navigation, purple and gray color palette, with notification badges and user avatar"
- "Clean SaaS dashboard interface, light theme, showing project metrics via donut charts and progress bars, top navigation bar, blue and white colors, card-based layout"

### Mobile App Screens
```
Mobile app [screen type] screen, [design style], [color gradient/scheme],
featuring [UI components], [interaction hints], [brand elements]
```

**Examples:**
- "Mobile app onboarding screen, modern flat design, purple to blue gradient background, featuring illustration of team collaboration, skip and next buttons, progress dots"
- "Mobile banking app home screen, clean minimal design, white background with green accents, featuring balance card, recent transactions list, bottom navigation"

### Landing Pages
```
[Industry] landing page hero section, [style] design, featuring [headline area],
[CTA buttons], [imagery type], [color scheme], [layout description]
```

**Examples:**
- "Tech startup landing page hero, modern glassmorphism design, featuring bold headline with gradient text, dual CTA buttons, 3D abstract shapes, dark background with neon accents"
- "E-commerce landing page, clean minimal design, featuring product showcase grid, search bar, category filters, white and coral color scheme"

## Marketing Material Patterns

### Social Media Posts
```
[Platform] post design for [purpose], [style] aesthetic, [dimensions context],
featuring [main visual], [text overlay area], [brand colors]
```

**Examples:**
- "Instagram post design for product launch, modern minimalist aesthetic, square format, featuring product on gradient background, bold typography area, pastel pink and white"
- "LinkedIn banner for tech company, professional corporate style, wide format, featuring abstract tech patterns, company tagline space, blue and silver"

### Banner Ads
```
[Size/format] banner ad for [product/service], [style], featuring [key visual],
[CTA button], [headline space], [brand guidelines]
```

**Examples:**
- "Leaderboard banner ad for fitness app, energetic dynamic style, featuring person exercising, 'Start Free Trial' CTA, bold motivational headline, orange and black"
- "Skyscraper banner for cloud service, professional tech style, featuring server illustration, 'Learn More' CTA, clean typography, blue gradient"

## Illustration Patterns

### Character Illustrations
```
[Character description] illustration, [art style] style, [pose/action],
[expression/mood], [color palette], [background context]
```

**Examples:**
- "Young professional woman illustration, flat vector style, sitting at laptop, focused expression, warm earth tones, minimal home office background"
- "Friendly robot mascot illustration, cute cartoon style, waving gesture, cheerful expression, blue and silver colors, transparent background"

### Scene Illustrations
```
[Scene description] illustration, [art style], [time of day/lighting],
[atmosphere/mood], [color scheme], [level of detail]
```

**Examples:**
- "Cozy coffee shop interior illustration, isometric style, warm afternoon lighting, relaxed atmosphere, brown and cream colors, detailed with plants and books"
- "Futuristic city skyline illustration, neon cyberpunk style, night scene, dramatic atmosphere, purple and teal neons, highly detailed"

## Logo Design Patterns

### Abstract Logos
```
Abstract [industry] logo, [style] design, [shape description], [color scheme],
[symbolism], suitable for [use case]
```

**Examples:**
- "Abstract tech company logo, modern geometric design, interconnected hexagons forming network pattern, blue gradient, symbolizing connectivity, suitable for digital platforms"
- "Abstract wellness brand logo, organic flowing design, leaf and wave fusion shape, green and teal, symbolizing nature and balance, suitable for print and digital"

[Company name] wordmark logo, [typography style], [letter treatment],
[color scheme], [style modifiers], clean vector design

**Examples:**

- "Bloom wordmark logo, elegant script typography, flourish on B, rose gold color, feminine luxury aesthetic, clean vector design"

## Icon Set Patterns

### UI Icons
```
Set of [number] [category] icons, [style] style, [stroke weight/fill type],
[color specification], [size context], consistent design language
```

**Examples:**
- "Set of 12 e-commerce icons, line art style, 2px stroke weight, single color black, 24x24px, consistent rounded corners"
- "Set of 8 finance icons, filled style, dual-tone blue shades, 32x32px, consistent geometric construction"

## Best Practices by Category

### For High Fidelity Mockups
- Specify exact UI components needed
- Include realistic content placeholders
- Mention device frame if needed
- Describe interaction states

### For Marketing Materials
- Include brand color codes if known
- Specify exact dimensions
- Mention target platform requirements
- Include space for required text elements

### For Illustrations
- Reference specific art styles
- Describe lighting and atmosphere
- Specify character details if people included
- Mention transparency needs for backgrounds
