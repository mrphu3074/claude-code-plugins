# Data Analysis Template

```xml
Analyze the following dataset and provide actionable insights.

<context>
{{DESCRIBE_BUSINESS_CONTEXT}}
</context>

<dataset format="{{csv|json|text}}">
{{YOUR_DATA}}
</dataset>

<analysis-goals>
1. {{PRIMARY_GOAL}}
2. {{SECONDARY_GOAL}}
3. {{TERTIARY_GOAL}}
</analysis-goals>

<instructions>
1. Identify key patterns and trends
2. Highlight anomalies or outliers
3. Provide statistical summary where relevant
4. Connect findings to business implications
5. Suggest actionable next steps
</instructions>

<output-format>
<summary>
[Executive summary: 2-3 sentences]
</summary>

<key-findings>
[Main discoveries with supporting data points]
</key-findings>

<anomalies>
[Unusual patterns requiring attention]
</anomalies>

<recommendations>
[Actionable next steps with priority]
</recommendations>
</output-format>
```

## Usage
Replace placeholders with your actual data and analysis goals.
