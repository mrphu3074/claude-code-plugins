# Document Analysis Template

```xml
Analyze the following document for {{PURPOSE}}.

<document type="{{DOCUMENT_TYPE}}">
{{DOCUMENT_TEXT}}
</document>

<context>
Purpose: {{ANALYSIS_PURPOSE}}
Audience: {{WHO_NEEDS_THIS}}
Focus areas: {{KEY_AREAS}}
</context>

<instructions>
1. Summarize key points
2. Identify important details in {{FOCUS_AREAS}}
3. Note any concerns or issues
4. Provide actionable recommendations
</instructions>

<output-format>
<summary>
[Brief overview of the document]
</summary>

<key-points>
[Main takeaways, bulleted]
</key-points>

<details>
[Detailed analysis of focus areas]
</details>

<concerns>
[Issues or risks identified]
</concerns>

<recommendations>
[Suggested actions]
</recommendations>
</output-format>
```

## Usage
Works for contracts, reports, policies, articles, etc.
Customize focus areas for your specific needs.
